Полный линейный прототип (без Node).
Запуск:
1) Распакуйте архив.
2) Запустите start_server_windows.bat
3) Откройте http://localhost:8000/

GitHub Pages:
- Загрузите содержимое архива в корень репозитория (index.html, src/, assets/).
- Settings → Pages → Deploy from branch → main / (root).


Если видите 404 на старые файлы (KhaosanScene.js, Dialogues.js) — это кеш браузера. Откройте в режиме инкогнито или сделайте Ctrl+F5.
